## Java Application to Calculate the monthly payment based on best interest rates


## Running the application

If you want to run the application locally you’ll need to follow these steps:

1) Clone it 
2) run ```mvn clean compile assembly:single```
3) java -cp  <PATH_TO_>ratecalculation-0.0.1-SNAPSHOT-jar-with-dependencies.jar com.zopa.ratecalculation.Main <PATH_TO>csv <LOAN_AMOUNT>

Example:
java -cp  C:\code\zopa\ratecalculation-0.0.1-SNAPSHOT-jar-with-dependencies.jar com.zopa.ratecalculation.Main C:\code\zoopa\lender.csv 10000
